### 观前提示
> ⚠**小桜帆**是 **闭源** 的机器人，请勿将源码以 **任何形式** 给他人。
> <br>
> （PS：但小桜帆的**功能性文档**是**公开可见的**）

### 历史
> **2021-2022年**，小桜帆的前身**ReinaBot**正式诞生
> <br>
> （PS：此时使用的几乎全是他人的插件，没有自主性）
>
> **2023年8-9月份**，**小桜帆**携着**鹿态V1.0**正式诞生
> <br>
> ~~（PS：这一时期 [鹿乃](https://zh.wikipedia.org/wiki/%E9%B9%BF%E4%B9%83) 因为**某些原因**开始暂停音乐活动，主要以桜帆鹿的形象进行游戏直播）~~
> 
> **2023年10月**，由于 [go-cqhttp的兴衰](https://github.com/Mrs4s/go-cqhttp/issues/2471) ，**小桜帆**作为 **QQ官方频道机器人** 诞生。
> 
> **2024年**，由于QQ频道的屡次改版，**小桜帆开发团队**决定放弃**频道端**的机器人开发工作，为**小桜帆**适配 **[OneBotV11](https://github.com/botuniverse/onebot-11)** 协议。
>
> **2024年4月中下旬**，**小桜帆**携带着全新的**鹿态V2.0**归来，开始了新的 ~~自主化（草）~~ 道路
> <br>
> ~~（PS：2024年4月，[《Stereophonic / ステレオフォニック》](https://www.bilibili.com/video/BV1cE421K7Tf)发布，鹿乃重新归音乐舞台！)~~
> <br>
> ~~(双喜临门，**好耶！！！！！！！！！！！！！！！！！！！！！！！！！！** )~~
>
> 到**如今**，**小桜帆开发团队**也在不断地开发出**小桜帆**的新功能！

🎧B站音频转换

### 特点：

> #### 优势
>
> - [x] 使用ffmpeg处理音频
> - [x] 使用mutagen处理音乐标签
> - [x] 支持多种音质转换
> - [x] 多线程处理，不影响事件循环

> #### 可转换音质
> *MP3：*
> - [x] 64K
> - [x] 132K
> - [x] 192K
> 
> *FLAC：*
> - [x] Hi-Res
> - [x] 杜比全景声
